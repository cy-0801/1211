## nrm---源管理器

1. **安装**

> npm install -g nrm

> > 查看当前有哪些源--->nrm ls

2. **添加源**

> nrm add <registry> <url>
>
> registry 为源名(起个别名)
>
> url为源地址
>
> eg: nrm add cpm http://192.168.22.11:8888/repository/npm-public/ 

3. **切换源**

> nrm user <registry>

4. 删除源

> nrm del <registry>

> nrm del cpm
